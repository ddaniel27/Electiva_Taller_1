Misc page

Aqui se incluyen distintos elementos que no era posible enlazar en otro sitio. Sin embargo tendrán su respectiva explicación.


Video Tag

Un tag diseñado para incluir videos en las páginas web. Muy usado para mostrar contenido audiovisual en las páginas a pesar de que incluir un video embebido alojado en otras plataformas es bastante común también.


SVG tag

Scalable Vector Graphics (SVG) es un formato ampliamente usado para definir gráficos con funciones matemáticas con el fin de mantener su calidad a pesar de reescalar sus dimensiones. Por tanto, se usa el tag SVG para incluir este tipo de contenido.

Datalist tag

Un tag usado para tener una lista dropdown con valores predefinidos a un campo de input. Muy usado para dar distintas opciones al usuario por obvias razones. Este tag por si solo no elimina la posibilidad de los usuarios de ingresar valores distintos a los sugeridos por el desarrollador.
