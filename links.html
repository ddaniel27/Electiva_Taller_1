Los links a diferentes archivos o carpetas pueden ser catalogados en dos grupos:

  • Enlaces relativos
  • Enlaces absolutos
Su principal diferencia radíca en la forma como cada tipo de enlace maneja la ruta para encontrar el archivo o dirección que busca.

Enlace relativo

El enlace relativo es generalmente el más usado cuando se trata de encontrar direcciones o archivos dentro de la misma máquina o instancia de un servidor web. Se dice que es relativo ya que el punto de partida de busqueda de la dirección especificada es relativa al sitio donde se encuentra la acción que está haciendo dicha busqueda.

Un enlace relativo se ve de la siguiente manera: "directorio/archivo.html"

Enlace abosoluto

El enlace absoluto es el más usado para viajar entre sitios web ya que necesita especificar toda la dirección del lugar al cual se desea dirigir. Es por esto que se llama absoluta, ya que la busqueda no inicia desde el lugar relativo donde se llama la acción de búsqueda, sino por el contrario, dicha acción busca la dirección completa especificada.

Un enlace absoluto se ve de la siguiente manera: "https://www.ejemplo.com/pagina/index.html"
